# jQuery UI

[https://jqueryui.com/]

Download (custom with all components selected) and unzip. The images folder, jquery-ui.min.css, jquery-ui.theme.css and jquery-ui.min.js is needed.

In project folder create css and js subfolders. 

css folder:

- images folder (see above)
- jquery-ui.min.css
- jquery-ui.theme.css
- style.css (own css)

js folder:

- jquery-3.2.1.min.js (latest from jQuery website)
- jquery-ui.min.js
- app.js (own js)

Link the jQuery stylesheets above the main stylesheet and the jQuery scripts as usually.

To use any of the components refer to the demos and API documentation on the jQuery API website.
