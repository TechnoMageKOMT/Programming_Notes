# Placeholder styling

To style placeholder text text in input or other fields use the following pseudo-elements:

```SCSS
&::-webkit-input-placeholder {
  color: $color-grey-dark-2;
}

&::-moz-placeholder {
  color: $color-grey-dark-2;
}

&::-ms-input-placeholder {
  color: $color-grey-dark-2;
}

&::placeholder {
  color: $color-grey-dark-2;
}
```
