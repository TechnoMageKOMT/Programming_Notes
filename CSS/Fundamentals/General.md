## Centering an element

## 7 Real-world steps to a fully functional website

### Step 1: Define Your Project

1. Define the goal of your project
2. Define your audience
3. Design with your goal and audience in mind

### Step 2: Plan Out Everything

1. Plan your content: text, images, videos, icons, etc.
2. Start thinking about visual hierarchy (design around content; called content first approach)
3. Define the navigation
4. Define the site structure if it is a bigger project

### Step 3: Sketch Your Ideas Before You Design

1. Get inspired and think about your design
2. Get the ideas out of your head: sketch your ideas before you start designing
3. Make as many sketches as you want, but don't try to make them perfect
4. Never start designing without having an idea of what you want to build

### Step 4: Design and Develop Your Website

1. Design your website taking into consideration good design principles (covered in design section)
2. Do this using HTML and CSS with the browser open
3. Use sketches, content and planning decisions you've made in steps 1, 2, and 3

### Step 5: Optimisation

1. Optimise performance: site speed
2. Search engine optimisation (SEO)

### Step 6: Launch

1. Find a webserver to host your website

### Step 7: Site Maintenance

1. Monitor user behaviour and change website if necessary
2. Update your content regularly: blog

Ref: [https://www.newdesigngroup.ca/website-design/web-design-process-infographic/] for indepth discussion

## Setting up the fluid grid for responsive webdesign

Google ranks mobile friendly websites higher than non-responsive sites.

### Three ingredients to responsive webdesign

1. Fluid grid: All layout elements are sized in relative units, such as percentages, instead of absolute units like pixels
2. Flexible images: are also sized in relative units
3. Media queries: allow specification of different CSS style rules for different browser widths
